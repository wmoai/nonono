# 実装プランの作成

テンプレートに沿った実装プランを立てる

## プランのテンプレート

```
# <!-- ここにプランのタイトルを記述する -->

## 概要
<!--
ここに今回のゴールや背景、方針などを記述する
-->

## やること
<!--
ここに進め方を段階的に記述する
-->

## 開発フロー

1. **Red**: 今回実装するものが要求通りであることを確認できるテストを先に書き、テストを実行して失敗を確認する。
2. **Green**: テストが通るように実装を書く
3. **Refactor**: `/simplify` でリファクタリングを行う
4. **Review**: 新規サブエージェントに `/code-review` を実行させる。指摘があれば Red からやり直す

## 検証

<!--
ここに検証方法について記述する
-->
```
